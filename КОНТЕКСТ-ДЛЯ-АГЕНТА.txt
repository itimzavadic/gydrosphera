ЗАЧЕМ ЭТОТ ФАЙЛ
---------------
Краткая сводка решений и устройства сайта. Полный чат в IDE не переносится;
исходное ТЗ — в файле «описание-лендинга-ГидроСфера.txt» (часть пунктов
устарела относительно текущей вёрстки — см. раздел «Фактическое состояние»).

ЧТО ЭТО ЗА ПРОЕКТ
-----------------
Статический лендинг компании ГидроСфера (бурение скважин, благоустройство
и ввод по всей Беларуси). Без сборщика: HTML + CSS + JS в каталоге public/.

СТРУКТУРА ФАЙЛОВ
----------------
public/
  index.html       — разметка, тексты, пути к изображениям
  css/styles.css   — стили (тёмная тема, неон, шапка, галерея, блоки)
  js/main.js       — бургер-меню, лайтбокс галереи, счётчик статистики,
                     класс is-scrolled у шапки при прокрутке
public/images/     — фото; есть симлинки logo.jpeg, blago-1/2 → кириллические имена

ЗАПУСК ЛОКАЛЬНО
---------------
  cd public && python3 -m http.server 8080
Открыть http://localhost:8080 (так надёжнее, чем file://).
